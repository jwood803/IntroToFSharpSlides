What is F#?

Functional first .NET language.

Mainly used for scientific and financial institutions.

Why F#?

  • Type providers
  • Option types
  • Rapid prototyping with the F# Interactive
  • Fewer bugs
    • Immutable by default
    • Units of Measure
  • Interoperability
    • Other languages than just .NET languages
        • R, Python, MATLAB
